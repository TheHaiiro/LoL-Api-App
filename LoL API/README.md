Praca Zaliczeniowa - Przeglądarka Bohaterów LoL
Opis Aplikacji
Jest to natywna aplikacja na platformę Android, która pełni rolę przeglądarki bohaterów z gry League of Legends. Aplikacja pozwala na przeglądanie listy wszystkich postaci, filtrowanie jej za pomocą wyszukiwarki oraz wyświetlanie szczegółowych informacji o wybranym bohaterze.
Projekt został zbudowany z myślą o nowoczesnych praktykach programowania na Androida, w tym:
•
Architektura MVVM (Model-View-ViewModel) z warstwą Repozytorium.
•
Obsługa trybu offline dzięki lokalnej bazie danych (Room).
•
Asynchroniczne pobieranie danych z sieci (Retrofit & Coroutines).
•
Nowoczesny i spójny interfejs użytkownika.
Użyte API
Aplikacja korzysta z oficjalnego API udostępnianego przez Riot Games:
•
Data Dragon API: https://developer.riotgames.com/docs/lol#data-dragon
Instrukcja Uruchomienia
Projekt jest standardowym projektem Android Studio. Aby go uruchomić:
1.
Sklonuj repozytorium na swój dysk lokalny.
2.
Otwórz projekt w najnowszej wersji Android Studio.
3.
Poczekaj, aż Gradle automatycznie zsynchronizuje wszystkie zależności.
4.
Uruchom aplikację na emulatorze lub fizycznym urządzeniu z systemem Android.
